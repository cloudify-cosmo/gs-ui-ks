Blueprint Arrows

Options

blueprint:arrows array
Assign the coordinates of X,Y of each source and target elements (in this case, the circles).
colors string
RGB or HEX color.
arrow boolean
Choose if to display the arrow head.
storke number
Size of the line by pixels
