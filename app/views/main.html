GS UI Components

Reusable Components for the GigaSpaces Management Console

Getting Started

Use this library by adding it to your bower in 1 command 

bower install https://github.com/guy-mograbi-at-gigaspaces/gs-ui-infra.git --save
You may refer to a specific tag / version by appending it to the url for example 
bower install https://github.com/guy-mograbi-at-gigaspaces/gs-ui-infra.git~0.1 --save
and then run bower install
