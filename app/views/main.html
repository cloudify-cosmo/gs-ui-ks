Getting Started

Dependencies

This library relies on , and {{ d.name }} for its various components. Additional dependencies are required per component. Add them as necessary.

Installation

Use this library by adding it to your bower with this command: 

bower install https://github.com/guy-mograbi-at-gigaspaces/gs-ui-infra.git --save

You may refer to a specific tag / version by appending it to the URL, for example: 

bower install https://github.com/guy-mograbi-at-gigaspaces/gs-ui-infra.git~0.1 --save

You can also update bower.json to include the library's entry:

{ "dependencies": { ... "gs-ui-infra": "git://github.com/guy-mograbi-at-gigaspaces/gs-ui-infra.git" } }

… and then run: 

bower install

Setup

Include the library's module in your app dependencies:

angular.module('myApp', ['gsUiInfraApp'])

Load the main module in your index.html:

<script src="bower_components/gs-ui-infra/app/scripts/app.js"></script>
