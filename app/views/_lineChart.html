Line Chart

Attributes Options

data array
The data of the line chart.
showLegend boolean
Legend display, true to make it visiable and false to hide it
color array / Function
The colors set of the lines
legendColor array / Function
The colors set of the legend points
tooltips boolean
If to display a tooltips or not
xAxisTickFormat Function
With this function you can manipulate the way you want to display the X Axis.
yAxisTickFormat Function
With this function you can manipulate the way you want to display the Y Axis.
